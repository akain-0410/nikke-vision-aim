# MarbleAim V6

Windows 10/11 弹珠轨迹辅助工具。程序只读取指定游戏窗口的画面，通过透明、鼠标穿透的覆盖层显示最佳落点和预测轨迹；不读取游戏内存，不控制鼠标，也不会自动发射。

本项目基于 **[kevin000077/nikke](https://github.com/kevin000077/nikke)** 二次开发，是非官方社区衍生版本。

## 主要功能

- 使用 Win32 物理像素定位与 Qt 坐标映射显示透明覆盖层。
- 16:9 全屏画面按 3840×2160 实测区域等比缩放；其他窗口使用视觉边框检测回退。
- 在六列网格中识别砖块，并过滤水纹、发射箭头和覆盖线等误检。
- 按实际飞行弹珠尺寸进行连续碰撞检测和多次反弹模拟。
- 只搜索正前方不到 180 度的可发射角度，碰撞次数限制为 5–10 次。
- 第 2–7 行危险权重逐层指数增加，优先处理接近第 8 行失败区的砖块。
- 高级模拟可选识别砖块 HP 和弹珠数量，并在砖块被击破后重算剩余弹珠路径。

高级模式默认关闭。关闭时完全跳过数字 OCR，只执行位置、生存权重和单球几何评分，以保持较低性能开销。

## 源码安装

当前只提供源码。要求 64 位 Python 3.12：

```powershell
python -m venv .venv
.\.venv\Scripts\python.exe -m pip install --upgrade pip
.\.venv\Scripts\python.exe -m pip install -e ".[test]"
```

## 启动

直接启动并从列表选择游戏窗口：

```powershell
.\.venv\Scripts\python.exe -m marble_aim
```

也可以指定窗口标题关键字：

```powershell
.\.venv\Scripts\python.exe -m marble_aim --window-title "游戏标题"
```

首次使用可打开调试框并执行校准：

```powershell
.\.venv\Scripts\python.exe -m marble_aim --window-title "游戏标题" --debug-view --calibrate
```

## 热键与控制

- `F7`：显示或隐藏覆盖层。
- `F8`：暂停或继续识别。
- `F9`：开始 20 秒校准；期间完成 3–5 次向左、中、右的发射。
- `F10`：调整弹珠半径、反射角、发射点和调试框。
- `重启识别`：重新识别外框、砖块、球半径和发射点。
- `选择窗口`：切换要捕获的游戏窗口。
- `模式 A/B/C/D`：循环切换轨迹显示方式。
- `高级 OFF/ON`：切换 HP OCR、弹珠数量识别和多球动态消块模拟。
- `退出助手`：保存配置、释放截屏资源并结束程序。

轨迹模式：

- A（默认）：绿色最佳落点＋紫色当前预测轨迹。
- B：仅绿色最佳落点。
- C：绿色最佳落点＋绿色完整推荐轨迹＋紫色当前预测轨迹。
- D：绿色最佳落点＋绿色完整推荐轨迹。

调试颜色：

- 青色实框：完整视觉识别区域。
- 紫色虚线：弹珠实际碰撞边界。
- 橙色虚线：开放底边。
- 绿色虚框：本回合锁定并参与碰撞计算的砖块。

## 识别与轨迹模型

16:9 全屏模式优先使用固定比例区域，避免动画和特效导致边界逐帧漂移；非 16:9 窗口才使用浅色四边检测。砖块定位使用六列规则网格，并同时检查主体边缘、深色轮廓和外圈支撑。

游戏瞄准圈大于实际飞行弹珠，因此物理半径不使用瞄准圈尺寸。根据实机截图，默认半径按砖块边长 `0.202` 计算；没有砖块时按碰撞框宽度 `0.032` 回退。在 3840×2160 全屏下约为 `31px`，其他尺寸同比缩放。

弹珠以圆形处理，砖块主体按直角正方形处理。上、左、右边界会按弹珠半径向内收缩，底边为开放出口。球心使用连续碰撞检测，并按碰撞法线反射：

```text
v' = v - 2(v·n)n
```

标准模式只使用位置和单球轨迹评分。高级模式开启后才加载数字模板，识别 1–3 位砖块 HP 和 `*XX` 弹珠数量；砖块 HP 归零后会从后续弹珠的场景中移除。无法可靠识别的砖块会保守地视为本轮不会消失。

## 分辨率与性能

- 以 3840×2160 的 16:9 实机画面为固定区域基准，其他 16:9 分辨率按比例缩放。
- 已针对 Windows DPI 缩放和 Qt 逻辑像素之间的映射进行修正。
- 窗口必须可见且不能最小化。
- 4K 截屏和高级模拟会增加 CPU 使用率，低配置设备建议保持 `高级 OFF`。
- 独占全屏或受保护渲染可能导致普通截屏得到黑屏，此时请改用窗口化或无边框窗口模式。

配置保存在：

```text
%LOCALAPPDATA%\MarbleAim\config.json
```

仓库中的 `config.example.json` 包含全部默认字段。删除个人配置文件可以恢复默认设置。
